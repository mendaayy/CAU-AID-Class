# CAU-AID-Class

## Usage
1. Run the script:
   ```bash
   python knn_iris_classifier.py
   ```
2. Enter the number of neighbors (k) when prompted.
3. View the model's accuracy and balanced accuracy
4. Input `sepal.length`

## Example
**Console Interaction**
```bash
Enter the number of neighbors for KNN:3
Model Accuracy: 0.9
Enter the sepal.length and petal.length: 5.1, 5.8
Predicted label: Setosa
```

## Notes
- Ensure that the input for predictions is numeric and follows format (`value1,value2`)
- Modify the dataset loading path
